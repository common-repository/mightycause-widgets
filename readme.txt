Integrates Mightycause fundraisers into your WordPress site.

== Description ==

Mightycause is a mission-driven, employee-owned business that has served the nonprofit community by helping them with online fundraising and donor engagement since 2006. This plugin allows you to embed Mightycause widgets on your WordPress site, making it easier for your visitors to donate to your cause.

**Note:** This plugin interacts with the Mightycause API to retrieve and display fundraising widgets on your website. By using this plugin, your site communicates with the Mightycause service. Please refer to their [Privacy Policy](https://www.mightycause.com/privacy) and [Terms of Use](https://www.mightycause.com/terms) for more information.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mightycause-integration/` directory, or install the plugin through the WordPress Plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the 'Settings' -> 'Mightycause' screen to configure the plugin.
4. Enter your Mightycause WordPress token on the settings page.

== Frequently Asked Questions ==

= How do I get a Mightycause WordPress Token? =

You can get your token from your Mightycause organization page, under Fundraising Tools > Widgets.

= Can I embed multiple widgets? =

Yes, you can embed multiple widgets using the Gutenberg blocks provided by the plugin.
